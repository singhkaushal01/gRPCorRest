Follow this:
--------------
1. Create a simple java application (no need to create spring boot application)
2. for pom dependencies and plugins required, refer here: https://github.com/grpc/grpc-java/blob/master/README.md
3. Create a proto file with following mandatory things:
   1. `syntax = "proto3";`
   2. use of multiple files `java_multiple_files = true;`
   3. Use of correct `java_package` 
4. Create `GrpcServer` class to start grpc server with service class `ProductController` added to it to process the 
   requests. e.g.
   ```
   GrpcServer server = new GrpcServer(_port_);
   BindableService productService = new ProductController();
   ServerBuilder.forPort(_port_).addService(productService);
   server = serverBuilder.build();
   server.start(); //abstract method to be implemented
   ```
5. Controller or Service class `extends` service implBase class generated by protoc
6. Override the default implementation of the service with custom implementation.
7. Postman(with `gRPC`) can be used to test the service after importing `.proto` file. 
8. This can be always useful: https://grpc.io/docs/languages/java/basics/
